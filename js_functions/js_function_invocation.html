JS function invocation

    A bit of info about invocation
  • 4 ways to do so
  • Each method will vary on how "this" is initialized
    Invoking a JS function
  • Remember the function (stuff inside) executes when it is called/invoked
  • Think of invoking and calling as being the same
  • Defining a function does not invoke it
  • In fact some say: call, invoke, start
    Invoking a function as a function
  • Remember the code block inside a JS function executes when something calls it
  • function myFunc(a, b){
      return a * b;
    }
  • myFunc(5, 6); // returns 30
  • myFunc does not belong to an object, however in JS there is always a default global object
  • In a browser the page object is the browser window
  • So the function will become a window function
  • myFunc() == window.myFunc()
  • Think of alert, location, etc
  • alert() == window.alert()
  • Don't use these type of calls though; think of conflicts and all
