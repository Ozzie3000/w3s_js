JS window Screen: get to know the user's screen

    window.screen
  • You can omit the window. just like window.alert()
  • screen.width; // returns width of screen not necessarily browser width // 1696 pixels
  • screen.height; // returns height of screen not necessarily browser height // 954
  • screen.availWidth; // 1696
  • screen.availHeight; // 912
  • screen.colorDepth; // 24
  • screen.pixelDepth; // 24
    Window Screen Width
  • From before the screen.width; returns size in pixels
    Window screen height
  • Height of user's screen in pixels
    Window screen available width
  • User's width in pixels minus any taskbar or other interface features
    Window screen available height
  • User's height in pixels minus any taskbar or other interface features
    Window screen color depth
  • Is the number of bits used to display just one color
