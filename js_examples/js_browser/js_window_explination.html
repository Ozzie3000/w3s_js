JS examples: window explained

    BOM
  • Browser Object Model
  • Allows JS to work with the browser
  • Currently no standards for it
  • Most browser (you can guess one that still acts up) use very similar JS implementations for interactivity
  • BOM is said to have methods and properties
    Window Object
  • It i supported by all browsers
  • Think of it as the browser window
  • All global JS functions, objects, variables automatically are members
  • Global functions: window object properties
  • Global variables: window object methods
  • The document object is a property of the window object
  • Example 1:
    window.document.getElementById("something");
    Same as:
    Example 2:
    document.getElementById("something");
  • Example 1:
    window.alert("Hello World!");
    same as:
    alert("Hello World");
    Window Size
  • There are 3 properties that you may use to determine the size of the browser window
  • We are speaking of the viewport; not counting toolbars or scrollbars
  • We will ignore 2 of them since I dislike IE
  • window.innerHeight - the inner height of the browser window
  • window.innerWidth - the inner width of the browser window
