JS Window | Browser Object model: talking to the browser

    BOM
  • No official standards... what?
  • Due to commonality of JS implementation accross browsers methods are gennerally referred to as BOM methods
    Window object
  • The window object is support by all browsers
  • All JS objects, variables, and functions become members of this object
  • Global variables are properties of the window object
  • You guessed it. Global functions are methods of the window object
  • The Document of the DOM is a property of the window object
  • window.document.getElementById("footer"); // same ass
  • document.getElementById("footer");
    Window sizes?
  • For Chrome, Firefox, newer versions of IE, Opera, Safari
  • window.innerHeight - the inner height of the browser window
  • Meaning the browser viewport
  • Does not include toolbars and scrollbars
  • For older versions of I.E. 8-5
  • document.documentElement.clientHeight
  • document.documentElement.clientWidth
  • or
  • document.body.clientHeight
  • document.body.clientWidth

    Other window methods
  • window.open(); // opens a new window or tab
  • window.close(); // closes current window or tab
