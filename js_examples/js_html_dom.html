JS HTML DOM

    DOM
  • JS and the Document Object model
  • JS can change all the HTML elements on the document object
  • JS can change all the HTML attributes on the page
  • JS can be used to change CSS
  • JS can be used to add new HTML elements and attributes
  • JS can be used to remove existing HTML element and attributes
  • JS can react to existing HTML events on a page
  • JS can be used to make new HTML events on a page
    Background info
  • DOM: Document object model
  • DOM is a standard
  • World Wide Consortium or W3C uses the DOM standard
  • DOM lays out standards for interacting with the document
  • DOM has three parts
    1. Core DOM - a model for all the variousw types of documents
    2. XML DOM - a model specifically for XML
    3. HTML DOM - a model specifically for HTML
    HTML DOM - defines
  • HTML elements as objects
  • properties of any HTML elements
  • methods availabe to accessing HTML elements
  • events for HTML elements
  • In a nutsell: HTML DOM is for creating, deleting, accessing, and changing HTML elements
