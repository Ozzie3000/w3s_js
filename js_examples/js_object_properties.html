JS object properties

    JS properties
  • Properties are the values associated with the JS object
  • Think of a JS object as a collection of properties
  • Most properties can added, changed, deleted
  • Some properties are read-only
    JS properties and various ways to access them
  • example 1:
    object_name.property;
    vehicle.make;
  • example 2:
    object_name["property"];
    vehicle["make"];
  • example 3:
    object_name[expression];
    var stuff = "make";
    vehicle[stuff];
    Using for...in loop
  • Used to loop through and object's properties
  • for (variable_name in object){
       code block
    }
  • the code block will be executed once per property in the object
  • var vehicle = {make:"ferrari", model:"enzo", year:2007, color:"red"};
  • var property_values = "";
  • var i; // doesn't seem to be necessary to declare this here
  • for (i in vehicle){
       property_values += vehicle[i] + " ";
    }
    Give me more properties
  • You may add new properties to objects
  • You may do so by giving it do so in a few ways
  • example 1: vehicle from above
    vehicle.transmission = "manual";
  • You cannot use reserved words for methods ore properties
    Property deletion
  • There is a keyword called delete you may use
  • example 1: vehicle from above
    delete vehicle.transmission; // deletes that property and its value and returns true
  • vehicle.transmission; // now returns undefined
  • The delete keyword is used for deletion of object properties
  • It will have no other effect on variables and functions
  • Do not be a troll and try to use it on baked in object properties unless you want to break stuff
    property attributes
  • Your properties will have names and values
  • A value is a property's attribute
  • More attributes: enumerable, configurale, writable
  • Attributes can describe how the property is accessed: readable, writable
  • All attributes are readable
  • Only the value is changable if the property is writable
  • ECMAScript 5 has methods for setting and getting all property attributes
