    prototype in more detail
  • prototype provides functions to help with programming HTML DOM
  • prototype has its $() function just like the popular jQuery
  • The $() function accepts HTML DOM element id values (or DOM elements)
  • it also adds new functionality to DOM objects
  • prototype has no ready() method to take the place of window.onload()
  • prototype adds extensions to the browser and the HTML DOM

    Event.observe()
  • Event.observe() method accepts 3 arguements

  • The event you are going to handle
  • The function you want to invoke
