JS syntax

    Deeper dive into syntax
  • Programs are just a set of instructions that can be executed by a computer
  • As you have learned those instructions are statements
  • JS is a programming language
  • JS uses semicolons to separate statements
    Keywords
  • The word var is a keyword
  • It creates a variable when used in a statement
  • var a = 5;
  • var b = 6;
    JS variables
  • Two types Global and local
  • x = 5; // global. Don't use this only an example
  • function someFunc(){ var y = 6; } // y is local to someFunc()
    JS statements
  • Statements are separated by semicolons
  • var birth = 3758;
  • var month = october;
  • JS statements can have: comments, expressions, keywords, operators, values
    JS numbers
  • Decimals are allowed
  • var num1 = 37.58;
  • var num2 = 3758;
    JS strings
  • var string1 = "I have double quotes";
  • var string2 = 'I have single quotes';
    JS expressions
  • You can make expressions and display the outcome
    JS operators
  • JS can use arithmetic operators
  • + - * /
  • var a = 5; var b = 6;
  • var c = a + b; // c == 11
    JS comments
  • Comments are ignored
  • //ignore just this line
  • /* ignore this line
    and this line
    and this line until you see this guy; the astrisk and forward slash */
    Case sensitivity
  • JS is case sensitive so be careful
  • var yoyoyo = "sing";
  • var yoyoYo = "sing"'
  • yoyoyo and yoyoYo are two different variables
    JS values - two types
  • Fixed values - known as literals
  • Variable values - known as variables
    JS literals
  • Fixed values
      numbers: 1, 40.5, 0.99 , etc
  • Strings are text wrapped in quotes
      "john baptist"
      'john baptist'
    JS variables
  • Variables are used for storing data
  • In JS the keyword var is used to define a variable
  • You use the equal sign to assign values on the right side to the left side (variable)
  • var a; // defined
  • a = 7; // assigned a value
