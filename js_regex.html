JS Regular expressions

    Regular expressions
  • sequence of characters (spaces to dots to numbers etc)
  • regular expressions are used to search for these patterns
  • /pattern/modifier
  • in JS: .search()
  • in JS: .replace()
    .search()
  • var str0 = "Hi my name is Mad Maxx";
  • /i at the end means case insensitive
  • var char1 = str0.search(/Mad/i); // 14 ; 14 where it starts
  • you can also use .search() with a string
  • var char2 = str0.search("Mad"); // 14
    .replace with regular expression
  • var str1 = "Hi you should be watching";
  • var char3 = str1.replace(/watching/i, "drinking");
    .replace() with a string
  • var str2 = "The great shinobi war 2";
  • var char4 = str2.replace("2", "3");
    regular expression modifiers
  • i : perform case-insensitive matching
  • g : perform a global match (instead of quiting at first match. lazy bum)
  • m : perform multiline matching
    regular expression patterns
  • [abcd] // looks for any of the characters between the brackets
  • [0-9] // looks for any digits between the brackets
  • (a|b) // find any either one divided by the |
    regex metacharacters
  • \d mean find a digit
  • \s mean find a white space
  • \b find a match at the beginning or at the end of a word
  • \uxxxx find the unicode character specified by the hexadecimal number xxxx
    quantifiers
  • n+ // matches any string that contains at least one n
  • n* // matches any string that contains zero or more occurrences of n
  • n? // matches any string that contains zero or one occurrences of n
    JS RegExp Object
  • object with predefined properties and methods
    test()
  • .test() method is a RegExp method
  • searches a string for a pattern, and returns true or false
  • var stpatty = /green/;
  • stpatty.test("It's all about the luck of the Irish and green everywhere"); // returns true
  • stpatty.test("It's all about the luck of the Irish and Green everywhere"); // returns false
  • stpatty2 = /green/i;
  • stpatty2.test("It's all about the luck of the Irish and Green everywhere"); // returns true
    .exec()
  • searches a string for a specified pattern, and if found returns the text
  • /y/.exec("It's all about the luck of the Irish and Green everywhere");
  • in Firefox Array["y"]
  • in Chrome ["y"]
  • /Y/.exec("It's all about the luck of the Irish and Green everywhere"); null
  • /Y/i.exec("It's all about the luck of the Irish and Green everywhere");
  • in Firefox Array["y"]
  • in Chrome ["y"]
  • when written to the screen it will simply show up as what you are looking for
