JS Regular expressions

    Regular expressions
  • sequence of characters (spaces to dots to numbers etc)
  • regular expressions are used to search for these patterns
  • /pattern/modifier
  • in JS: .search()
  • in JS: .replace()
    .search()
  • var str0 = "Hi my name is Mad Maxx";
  • /i at the end means case insensitive
  • var char1 = str0.search(/Mad/i); // 14 ; 14 where it starts
  • you can also use .search() with a string
  • var char2 = str0.search("Mad"); // 14
    .replace with regular expression
  • var str1 = "Hi you should be watching";
  • var char3 = str1.replace(/watching/i, "drinking");
