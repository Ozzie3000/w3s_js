swtich statement used to select one from many blocks of code
  • Switch(expression){
       case a:
        code block
        break;
       case b:
        code block
        break;
       default:     default codes block
    }
  • the heart of the matter
  • The switch expression is evaluated once
  • The value of the expression is compared to each case one at a time
  • If there is a match the matching block is executed
  • If no match occurs default is run
    common code and fall-through
  • in come situations you may want mulitple cases to use the same code.
  • in this event let them fall through and use a default code to run a block of code for those commonality
  • default does not have to be the last code block.
  • if default is not last remember to end it with a break
