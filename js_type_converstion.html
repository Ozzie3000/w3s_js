JS type conversion

    5 JS data types that contain values
  • string
  • number
  • boolean
  • object
  • function
    3 types of objects
  • Object
  • Date
  • Array
    2 data types without values
  • null
  • undefined
    examples
  • typeof "John"; // string
  • typeof 3.16; // number
  • typeof NaN; // number
  • typeof false; // boolean
  • typeof ["a","b","c","d"]; // object
  • typeof {name:"john", age:36} // object
  • typeof new Date(); // object
  • typeof function() {}; // function
  • typeof myStuff; // undefined (for undeclared)
  • typeof null; // object
    break down
  • data type NaN is a number
  • data type array is an object
  • data type date is an object
  • data type null is an object
  • data type undefined variable is undefined
    typeof
  • typeof is an operate that returns operands
  • can't distinguish between: objects, array, date
    constructor
  • "john".constructor // returns function String() { [native code] }
  • (3.16).constructor // returns function Number() { [native code] }
  • false.constructor // returns function Boolean() { [native code] }
  • [1,2,3,4].constructor // returns function Array() { [native code] }
  • {name:"john", age:316}.consructor // returns function Object() { [native code] }
  • new Date().constructor // returns function Date() { [native code] }
  • function(){}.constructor // returns function Function() { [native code] }
    checking objects for array-ness
  • function is_array(whatever){ return whatever.constructor.toString().indexOf("Array") > -1; }
  • var stuff = {name:"john", age:316};
  • is_array(stuff); // returns false
  • var peeps= ["john", "Matthew", "Peter"];
  • is_array(peeps); // returns true
    JS type conversion
  • JS variables can be converted to a new variable and another data type
  • by use of JS function
  • by use JS itself
  • examples
  • var a = 5; String(a); // returns string from number variable a. "5"
  • String(123); // "123"
  • String(100 + 23); // "123"
  • .toString() does the same thing
  • var b = 345; b.toString(); // "345"
  • (123).toString(); // "123"
