JS math object: Math has no constructor. No methods have to create a Math object first.

    Math.random()
  • Math.random(); // returns a random number 0-1
  • Math.random()*10 // returns a random number 0-10
    Math.min() and Math.max()
  • Math.min(11, 123, 321, 0, -18); // returns -18
  • Math.max(11, 123, 321, 0, -18); // returns 321
    Math.round()
  • Math.round(5.9); // returns 6
  • Math.round(3.4); // returns 3
    Math.ceil(); rounds up
  • Math.ceil(1.2); // returns 2
    Math.floor(); rounds down
  • Math.floor(1.9); // returns 1
    combos
  • Math.Floor(Math.random()*11); // returns 0-10
  • Math.ceil(Math.random()*10); // returns 1-10
    Math constants
  • Math.E; // Euler's number
  • Math.PI;
  • Math.SQRT1_2 // returns square root of 1/2
  • Math.SQRT2 // returns squre root of 2
  • Math.LN2 // returns the natural logarithm of 2 -> 0.6931471805599453
  • Math.LN10 // returns the natural logarithm of 10 -> 2.302585092994046
    Math Object methods
  • abs(x) // returns the absolute value of x
  • acos(x) // returns the arccosine of x, in radians
  • asin(x) // returns the arcsine of x, in radians
  • atan(x) // returns the arctangent of x as a numeric value between -PI/2 and PI/2 radians
  • atan2(y,x) // returns the arctangent of the quotient of its arguments
  • ceil(x) // returns x, rounded upwards to the nearest integer
  • cos(x) // returns the cosine of x (x is in radians)
  • exp(x) // returns the value of Ex
  • floor(x) // returns x, rounded downwards to the nearest integer
