    Tree, nodes, elements

  • Text inside those HTML elements are text nodes
  • HTML attributes are attribute nodes
  • Comments are [you guessed it] comment nodes
  • You can use JS to access all those nodes
  • You can also create nodes
  • You can modify nodes
  • You can delete nodes
