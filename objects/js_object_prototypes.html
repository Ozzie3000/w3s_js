JS object prototypes

    JS prototypes
  • Every JS object has a prototype
  • the prototype is also and object
  • JS objects inherit their properties and methods from their prototype
  • objects created usign an object literal, or with the new keyword (new Object() ) inherit from a prototype called Object.prototype
  • Objects create with the new Date() inherit the Date.prototype
  • the Object.prototype is on the top of the prototype chain
  • All JS objects (Date, Array, RegExp, Function, etc) inherit from the Object.prototype
    Create a prototype
  • Use the object constructor function
  • function person(firstName, lastName, age, eyeColor) {
      this.firstName = firstName;
      this.lastName = lastName;
      this.age = age;
      this.eyeColor = eyeColor;
      this.changeLastName = function (name) {
          this.lastName = name;
      }
    }
  • now you can use the new keyword to create new objects using the prototype
  • var vader = new person("anakin", "skywalker", 18, "blue");
  • var padme = new person("padme", "naberrie", 26, "brown");
  • vader.firstName; // returns "anakin"
  • padme.age; // returns 26
