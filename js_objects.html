JavaScript Objects

    Objects
  • objects have property:property value pairs, and methods
  • var car = {name_0:"value_0", name_1:"value_1", name_2:"value_2"};
  • var truck = {make:"Toyota", model:"Tundra", year:2020};
  • methods are stored as function definitions
  • store a method as a name:value pair:
    car_description: function() {return this.year + " " + this.model;}
  • var truck = {make:"Toyota", model:"Tundra", year:2020, description:function() {return this.year + this.model;}}
  • to call property:
    object_name.property_name or
    object_name["property_name"]
  • to call a method:
    object_name.method_name() or
    object_name[]
