JS array methods

    converting arrays to strings
  • in JS all objects have .valueOf() and .toString() methods
  • var fruits = ["apple", "banana", "clementine", "durian"];
  • in console fruit will return // ["apple", "banana", "clementine", "durian"]
  • document.getElementById("demo").innerHTML = fruits.valueOf(); // will write this --> apple, banana, clementine, durian
    use .join() method to combine all elements into a string
  • var fruits = ["apple", "banana", "clementine", "durian"];
  • fruits.join("" || ""); // returns "apple || banana || clementine || durian"
    give and take || .pop() .push() methods as last index
  • fruits.pop(); // takes out last element. changes array
  • fruits.push("durian"); // puts stuff between "" at the end. changes array
    shifting elements
  • .shift() takes away index 0.
  • .unshift() add element at index 0.
  • like pop and push array is changed
  • fruits.shift() // removes "apple" at index 0
  • fruits.unshift("apple") // add "apple" at index 0
    changing elements
  • fruits[0] = "Al"; // index 0 changed from apple to Al
  • append to last element with length
  • fruits[fruits.length] = "eggplant";
