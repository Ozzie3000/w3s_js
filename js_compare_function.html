The compare function

    Purpose of compare function
  • to define an alternative sort order
  • works by compare 2 arguments and return one of 3 values: negative, zero, positive
  • sorts from lowest to highest or highest to lowest depending on last statement
  • .sort(function(a,b){ return a-b}); // lowest to highest
  • .sort(function(a,b){ return b-a}); // highest to lowest
