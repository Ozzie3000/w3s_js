JS HTML DOM elements

    Finding elements
  • You can find elements by the following:
  • id
  • tag name
  • class name
  • CSS selectors
  • HTML object collections
    Fining HTML elements by id
  • var example1 = document.getElementById("unique1");
  • If something is found this method will return the element as an object in the variable
  • Otherwise the variable will contain null
    Finding HTML elements using tag name
  • var example2 = document.getElementByTagName("p");
  • If it finds multiple they get stored in a collection
  • You may access each one by index
  • You can also target a section then look for a tag in that section
  • var a = document.getELementById("left_column");
  • var b = a.getElementsByTagName("footer");
    Finding HTML elements by Class Name
  • getElementsByClassName()
  • var c = document.getElementsByClassName("container1");
  • This to returns a collection
  • x[0].innerHTML; grabs the inner part of the first one
