JS strings

    Storing info
  • You can store information using strings
  • Strings can either be created with "" or ''
  • var carname1 = "viper";
  • var carname2 = 'venom";
  • If you need to use a set inside another alternate
  • var quote1 = "My favorite car is 'the venom' if you know what that is";
    How much?
  • There is a built in property called length that can tell you this
  • var quote1 = "My favorite car is 'the venom' if you know what that is";
  • console.log(quote1.length); // shows 55
    Capturing the stubborn special ones
  • You are not going to listen to anyone and want to nest quotation marks inside other quotation marks
  • var example1 = "The standard \"hello world\" phrase";
  • alert(example1); will write "The standard "hello world" phrase";
