JavaScript String Methods

    indexOf(); returns location of fist occurance
  • var string_0 = "Hello, please have a seat. Would you please let someone know if you need help.";
  • var position_0 = string_0.indexOf("please");
  • returns 7
  • returns -1 if no match
    lastIndexOf(); returns the last location occurance
  • var string_1 = "Hello, please have a seat. Would you please let someone know if you need help.";
  • var position_0 = string_0.indexOf("please");
  • returns 37
  • returns -1 if no match
    search(); returns location of fist occurance; kind of like indexOf()
    can be more powerful than indexOf(). See regular expressions
  • var string_2 = "Hello, please have a seat. Would you please let someone know if you need help.";
  • var position_2 = string_2.search("plea");
  • returns 7
  • returns -1 if no match
    extracting string parts
    • slice()
    • var fruits = "apple, banana, cantalope, durian";
    • var portion_0 = fruits.slice(15,24);
    • portion contains "cantalope"
    • var portion_1 = fruits.slice(-6, -1);
    • portion_1 contains "duria"
    • var portion_2 = fruits.slice(-6) // no 2nd parameter will go till end
    • portion_2 contains "durian"
    • substring() similar to slice, but does not accept negative values
    • var fruits = "apple, banana, cantalope, durian";
    • var portion_3 = fruits.substring(15,24);
    • portion_3 contains "cantalope"
    • var portion_3 = fruits.substring(15); // no 2nd param so it will go to the end
    • portion_3 contains "cantalope, durian"
    • substr() similar to slice, but 2nd param length of extracted part
    • var fruits = "apple, banana, cantalope, durian";
    • var portion_4 = fruits.substr(-15,6);
    • portion_4 contains "ntalop"
    • var portion_4 = fruits.substr(15); // no 2nd param so it will go to the end
    • portion_4 contains "cantalope, durian"
    • replace() similar to slice, but 2nd param length of extracted part
    • this method can also take regular expressions
    • var fruits = "apple, banana, cantalope, durian";
    • var portion_5 = fruits.replace("cantalope","cranberry");
    • portion_5 contains "apple, banana, cranberry, durian"
    • fruits still contains "apple, banana, cantalope, durian"
    • Converting to UPPER and lower Case
    • var string_1a = "Hello everyone how's the weather?";
    • var string_1b = string_1a.toUpperCase();
    • string_1b contains: "HELLO EVERYONE HOW'S THE WEATHER?";
    • var string_2a = "GO HOME EVERYBODY!";
    • var string_2b = string_2a.toLowerCase();
    • sting_2b contains: "go home everyone!"
    • concat() method
    • var string_0 = "a";
    • var string_1 = "b";
    • var string_2 = string_0.concat("," , " " , string_1 , "," ," " , "c");
    • string_2 contains: "a, b, c"
    • **** All string methods return a new string. They don't modify the
      original string. Formally said: Strings are immutable: Strings cannot be
      changed, only replaced. ****
    • extracting string characters
    • charAt(position) // returns the character at that index in a string
    • charCodeAt(position) // returns the unicode of the character at that index in a string
      • var string_0 = "Welcome to Jurassica Park";
        string_0.charAt(8); returns "t"
      • var string_1 = "Welcome to Jurassica Park";
        string_1.charCodeAt(8); returns 116
