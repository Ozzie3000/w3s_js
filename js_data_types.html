JavaScript Data Types

    JavaScript Data Types examples
  • Primative data types
    • Number: var height = 120;
    • String: var name = "tacoma"
    • Boolean: var answer = true;
    • Array: var fruit = ["apple", "banana", "cantalope"]
    • Symbol: var symbol_1 = Symbol("stuff"), var symbol_2 = Symbol("stuff");, var symbol_3 = ("stuff").
      • ***The above code creates three new symbols. Note that Symbol("stuff") does not coerce the string "stuff" into a symbol. It creates a new symbol each time:
    • Null
  • Objects
