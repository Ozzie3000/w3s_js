JS number methods

    Global Methods
  • Number() returns a number, converted from its argument
  • parseFloat() parses its argument and returns a floating point number
  • parseInt() parses its argument and returns and integer
    Number methods
  • all number methods returns a new variable and do not change original
  • toString() returns a number as a string
  • toExponential() returns a string, with a number rounded and written using exponential notation
  • toFixed() returns a string, with a number rounded and written with a specified number of decimals
  • toPrecision() returns a string, with a number written with specified length
  • valueOf() returns a number as a number // var x = 34; x.valueOf(); returns 34
    toString() method
  • var x = 357;
  • x.toString(); // returns "357"
  • (357).toString(); // returns "357"
  • (1 + 23).toString(); returns "24"
    toExponential() method
  • var x = 1234567; //
  • x.toExponential(0); // returns 1e+6
  • x.toExponential(2); // returns 1.23e+6
  • x.toExponential(3); // returns 1.235e+6
  • x.toExponential(4); // returns 1.2346e+6
    toFixed() method
  • var x = 1.234567; //
  • x.toFixed(0); // returns 1
  • x.toFixed(2); // returns 1.23
  • x.toFixed(3); // returns 1.235
  • x.toFixed(4); // returns 1.2346
    toPrecision() method; arguments have to be between 1-21
  • var x = 1.234567;
  • x.toPrecision(); // returns 1.234567
  • x.toPrecision(21); // returns 1.23456699999999996997
  • x.toPrecision(1); // returns 1
  • x.toPrecision(2); // returns 1.2
  • x.toPrecision(8); // returns 1.2345670
