JavaScript Scope

    JS scope
  • In JavaScript, objects and functions, are also variables.
  • In JavaScript, scope is the set of variables, objects, and functions you have access to.
  • JavaScript has function scope: The scope changes inside functions.
    Local variables
  • only available to function (local to that function)
      function a_function(){ var car_name = "Enzo"; }
      variable car has local scope
    Global variables: (various ways of doing this)
  • var a=0; //1 (outside a function)
  • a=0; //2 (in or outside a function)   stay away from this style
  • window.a=0; //3
  • this.a=0; //4
