JS popups - yay...

    JS popups
  • Alert box
  • Confirm box
  • Prompt box
    Alert box
  • A way or reminding users of something or bring it to their attention
  • The user will need to select the "ok" button to continue
  • window.alert("stuff goes here");
  • alert("stuff goes here"); // window omitted
    Confirm box
  • Used to get user to verify something or accept it
  • User will get a "ok" and "cancel" options, but will need to select one
  • When "ok" is selected true is returned
  • When "cancel" is selected false is returned
  • window.confirm("stuff also goes here");
  • confirm("stuff also goes here"); // shorter version
    Prompt box
  • Used to get input from the user
  • User will enter a value
  • User will then get a "ok" and "cancel" options, but will need to select one
