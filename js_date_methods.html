JS date methods

    Date get methods
  • var now = new Date()
  • getDate() // get day as a number (1-31)
  • getDay() // get weekday as a number (0-6)
  • getFullYear() // get the four digit year (YYYY)
  • getHours() // get hour (0-23)
  • getMilliseconds() // get the millis (0-999)
  • getMinutes() // get the mintues (0-59)
  • getMonth() // get the month (0-11)
  • getSeconds() // get the seconds (0-59)
  • getTime() // get the time (milliseconds since jan 1 1970)
  • now.getFullYear() --> 2015
    Date set methods
  • var now = new Date()
  • setDate() // set day as a number (1-31)
  • setFullYear() // set the four digit year (YYYY)
  • setHours() // set hour (0-23)
  • setMilliseconds() // set the millis (0-999)
  • setMinutes() // set the mintues (0-59)
  • setMonth() // set the month (0-11)
  • setSeconds() // set the seconds (0-59)
  • setTime() // set the time (milliseconds since jan 1 1970)
  • now.setFullYear(2015) --> 2015
