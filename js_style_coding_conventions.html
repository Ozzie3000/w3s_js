  • use white space around operators

    Code indentation
  • People generally use 4 spaces to indent code blocks
  • Stay away from using tab. Varies from editor to editor in the way it gets interpretted
    Statement rules
  • end simple statements with a semicolon
  • var aGuy = "Abraham";
  • var firstPerson = {firstName: "Abraham", homeTown: "Canaan", eyeColor: "silver"];
  • for complex statements
  • use { at the end of the first line
  • use one white space before opening bracket
  • when finishing close with } on last line with no leding white space
  • omit the semicolon
