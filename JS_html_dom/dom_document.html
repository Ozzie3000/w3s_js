JS HTML DOM document

    The HTML DOM Document
  • Basically the object that represents your web page
  • This document object owns all other objects on the web page
  • To access things you must first start by accessing the document object first
    Search and find HTML elements
  • Method   Description  
  • document.getElementById() Looks for an element with that id
  • document.getElementsByTagName() Looks for elements by tag name
  • document.getElementsByClassName() Looks for elements by class name
    Mix it up and change stuff
  • Method   Description  
  • element.innerHTML= Changes the inner HTML of target element
  • element.attribute= Changes the attribute HTML of target element
  • element.setAttribute(attribute,value) Changes the attribute HTML of target element
  • element.style.property= Changes the style of an HTML element
    Elements: adding and deleting them
  • Method   Description  
  • document.createElement() Creates an HTML element
  • document.removeChild() Removes an HTML element
  • document.appendChild() Add an HTML element to something
  • document.replaceChild() Replace an HTML element
  • document.write(textgoeshere) Writes into the HTML output stream. Try not to use this other better methods
    Adding event handlers
  • Method   Description  
  • document.getElementById(id).onclick=function(){ code block } Ex. of adding an event handler code to an onclick event
