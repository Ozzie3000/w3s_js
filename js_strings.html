    Escape characters
  • \' : single quote
  • \" : double quote
  • \\ : backslash
  • \n : new line
  • \r : carriage return
  • \t : tab
  • \b : backspace
  • \f : form feed
    Breaking long lines of code
  • \ symbol inside a string, but not ECMAScript standard
  • after an operator like = , + , etc
  • \ not allowed else where
    String properties and methods
  • Primitive values do not have properties and methods
  • JS treats strings like objects when executing properties and methods
      • Properties
    • constructor: returns the function that created the string object's prototype
