JS Variables 2

    JavaScript Identifiers or known as unique names
  • Names can contain letters, digits, underscores, and dollar signs.
  • Names must begin with a letter
  • Names can also begin with $ and _ (this convention is used in jQuery. more frequently seen there)
  • Names are case sensitive (a and A are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as names
    JavaScript Data types
  • Six data types that are primitives:
    • Boolean
    • Null
    • Undefined
    • Number
    • String
    • Symbol (new in ECMAScript 6)
  • Object
    Declaring JS variables
  • Declare with empty value: var a;
  • pass it a value: a = 2;
  • Declare with value: var b = 3;
  • Declare multiple variables in one line: var apple = 3.2, banana = 2.4, cantalope = 4.9;
