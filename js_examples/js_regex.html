JS Regular expressions

    Background, intel on regexp
  • Think of it as a language used for finding whatever you are looking for
  • You can be as vague or insanely specific as you want to
  • The sequence of characters will make up your search pattern
  • The pattern will describe what you are looking for
  • It can also be used to match and replace
    Syntax
  • /pattern/modifiers;
  • example: var pattern1 = /hello/i;
  • match hello
  • i means case-insensitive
    strings
  • search = search()
  • replace = replace()
  • search returns the position of a match
  • replace returns the modification where the replacement occurred
  • var stuff1 ="aow3r4 owaioigr HEllo world oaewrijloioj";
  • var match1 = stuff1.search(/hello/i);
  • match1 contain 16
  • var match2 = stuff1.search("HEllo");
  • match2 conatins 16, but we had to be case sensitive
    String replace() with a regular expression
  • var pattern2 = "aow3r4 owaioigr HEllo world oaewrijloioj";
  • var reggy_replace = pattern2.replace(/hello/i, "weh woh");
    String replace() using a string
  • var pattern3 = "aow3r4 owaioigr HEllo world oaewrijloioj";
  • var match3 = pattern3.replace("HEllo world", "you found me");
  • match3 contains "aow3r4 owaioigr you found me oaewrijloioj"
