JS array methods

    converting arrays to strings
  • in JS all objects have .valueOf() and .toString() methods
  • var fruits = ["apple", "banana", "clementine", "durian"];
  • in console fruit will return // ["apple", "banana", "clementine", "durian"]
  • document.getElementById("demo").innerHTML = fruits.valueOf(); // will write this --> apple, banana, clementine, durian
    use .join() method to combine all elements into a string
  • var fruits = ["apple", "banana", "clementine", "durian"];
  • fruits.join("" || ""); // returns "apple || banana || clementine || durian"
    give and take || .pop() .push() methods as last index
  • fruits.pop(); // takes out last element. changes array
  • fruits.push("durian"); // puts stuff between "" at the end. changes array
    shifting elements
  • .shift() takes away index 0.
  • .unshift() add element at index 0.
  • like pop and push array is changed
  • fruits.shift() // removes "apple" at index 0
  • fruits.unshift("apple") // add "apple" at index 0
    changing elements
  • fruits[0] = "Al"; // index 0 changed from apple to Al
  • append to last element with length
  • fruits[fruits.length] = "eggplant";
    deleting
  • delete fruits[fruits.length]; that index (last one) will not be undefined
    splicing an array
  • name_of_array.splice(i, i, "element_1", "element_2"); fist i = index location to start in, second i number of elements to remove
  • fruits.splice(2, 0, "test1", "test2");
  • fruits not looks like this: "apple", "banana", test1", "test2", clementine", "durian"
    sorting arrays
  • fruits now contains --> "apple", "banana", "test1", "test2", "clementine", "durian"
  • fruits.sort() --> "apple", "banana", "clementine", "durian", "test1", "test2"
    reversing an array
  • var fruits = ["apple", "banana", "clementine", "durian", "test1", "test2"]; // from previous example
  • fruits.reverse(); --> ["test2", "test1", "durian", "clementine", "banana", "apple"]
    numeric sort w/ compare function
  • .sort() does not work well for integers so i needs a helper
  • var nums = [992, 355, 111, 321, 987];
  • nums.sort(function(a,b){return a-b}); changes array
  • nums now --> 111, 321, 355, 987, 992
    reverse numerica sort with compare function
  • var nums = [992, 355, 111, 321, 987]; // reset
  • nums.sort(function(a,b){b-a}; // changes array
  • nums now --> 992, 987, 355, 321, 111
