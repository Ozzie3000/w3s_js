    JS assignment operators
  • Operator       Example       Twin brother
  • =         a = b         a = b
  • +=       a += b     a = a + b
  • -=        a -= b       a = a - b
  • *=        a *= b       a = a * b
  • /=        a /= b       a = a / b
  • %=        a %= b       a = a % b

    JS comparison and other operators
  • Operator       Description
  • ==       check for loose equality

  • !=       checks for not equality
  • !==       checks for not equality in type or value
  • >       greater than
  • >=       greater than or equal to
  • <       less than
  • <=       less than or equal to
