JS date formats

    Valid JS date strings
  • ISO dates
  • Long dates
  • Short dates
    JS ISO dates
  • ISO 8601 syntax (YYYY-MM-DD) is the newest and preferred JS format
  • Month is written before day in all short date and ISO date formats.
  • var date_0 = new Date("2015-1-1"); // Thu Jan 01 2015 00:00:00 GMT-0500
  • strangeness:
    var date_1 = new Date("2015-1-1"); // Wed Dec 31 2014 19:00:00 GMT-0500
  • var date_1 = new Date("2015-03-25T12:00:00"); // Wed Mar 25 2015 08:00:00 GMT-0400 // T indicates UTC time
    JS Long dates
  • often written as: "MMM DD YYYY"
  • var stuff_0 = new Date("Mar 10 2015"); // Tue Mar 10 2015 00:00:00 GMT-0400
  • var stuff_1 = new Date("2015 Mar 10"); // Tue Mar 10 2015 00:00:00 GMT-0400
  • var stuff_1 = new Date("2015, March, 10"); // Tue Mar 10 2015 00:00:00 GMT-0400
  • var stuff_1 = new Date("2015 March 10"); // Tue Mar 10 2015 00:00:00 GMT-0400
    JS Short Dates
  • often written as MM/DD/YYYY or MM-DD-YYYY js will also accept "YYYY/MM/DD"
  • var stuff_2 = new Date("01/02/3456"); // Wed Jan 02 3456 00:00:00 GMT-0500
  • var stuff_3 = new Date("06-05-4321"); // Sun Jun 05 4321 00:00:00 GMT-0400
  • var stuff_4 = new Date("1234/05/06"); // Sat May 06 1234 00:00:00 GMT-0400
