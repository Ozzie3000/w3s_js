JS libraries

    JavaScript frameworks (libraries)
  • Programming using advanced (such as handling of complex nuances and or giant dissimilaraties between browsers) is exceeding difficult, time-consuming, and complex to work with.
  • To alleviate the problems developers came out with libraries to help
  • Sometimes these libraries are referred to as frameworks
  • jQuery
  • Prototype
  • Mootools
  • Commonalities among them are as follows: they all have functions for common JavaScript tasks like animations, DOM manipulation, and Ajax handling.
    jQuery
  • Is the most popular framework around today
  • If you like CSS this is for you!
  • CSS selectors are used to access and manipulate DOM objects on a web page.
  • jQuery has a companion UI framework and various plug-ins
  • Who uses jQuery?
  • Only: Google, Netflix, IBM, Microsoft
    Prototype
  • Prototype is a JS library which provides a simple API to perform common web tasks
  • API - Application Programming Interface
  • Later on you will see APIs can be written in various languages
  • API - is a library of properties and methods for manipulating the HTML DOM
  • Prototype expands JS by providing classes and inheritance
    Mootools
  • MooTools - another framework
  • It offers an API to make common JS programming easier
  • It also includes some light weight effects and animation
    Misc frameworks
  • Dojo - a toolkit for DOM events, manipulation, widgets, and some other hocus pocus
  • Ext JS - Used for building internet applications in the form of widgets that are customizable
  • script.aculous - Open source JS framework used for interface behaviors and visuals effects
  • UIZE - AJAX, DOM, templates, Widgets, and more hocus pocus
