JS variables

    JS variables
  • Think of them as buckets that hold stuff
  • They act as containers for values when data is stored in them
  • var fruit1 = "apple";
  • var fruit2 = "banana";
  • var snacks = fruit1 + ", " + fruit2;
  • alert(snacks);
  • all three of those are variables that are really just containers for values
    JS identifiers
  • All variables must be indentified
  • Unique names must be identified
  • Since JS is a weakly typed langauge if you do not it will overwrite it with a new value
  • Names must begin with a letter
  • You can also use _ and $ to begin the name, but stay way from this
  • Names can contains digits, dollar symbols, letters, and underscores
  • Names are case sensitive
  • Do not use reserved words like : var, return, etc
