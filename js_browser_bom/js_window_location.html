Window JS location: window.location object can get the current page (URL) and to redirect the browser to a new URL.

    Window location
  • window.location object can omit the window prefix
  • window.location.href; // returns href (URL) of current page
  • window.location.hostname; // returns domain name of web page
  • window.location.pathname; // returns path and filename of the current page
  • window.location.protocol; // returns the web protocol used (http:// or https://)
  • window.location.assign; // loads a new document
    Window location href example
  • document.getElementById("ref").innerHTML = "Page location is " + window.location.href;
    Window location hostname
  • document.getElementById("demo").innerHTML = "Page hostname is " + window.location.hostname;
