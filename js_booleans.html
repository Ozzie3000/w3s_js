JS Booleans

    Boolea values
  • in JS boolean represents: true or false
  • but often you need things like: yes, no, on, off, true, false
    using the Boolean function()
  • Boolean() to see if an expression or variable is true:
  • Boolean(2>1); // returns true
  • Boolean("tr">"st"); // returns true
    comparisons and conditions
  • truthy or falsy
  • == compare to see if loose equality. example: if(day == "Friday")
  • > compare to see if greater than. example: if(salary > 1234)
  • < compare to see if less than. example: if(salaray < 1234)
    things that eval to true
  • Boolean(var_here) -> true; // var_here can be the following:
  • Boolean(1000); // number
  • Boolean(3.14); // float
  • Boolean(-2); // negative number
  • Boolean("anyString"); // string
  • Boolean(1+2+3); // no zero expressions
  • Boolean(true);
    things that eval to false
  • Boolean(0); // returns false
  • Boolean(-0); //
  • Boolean(""); // empty string
  • Boolean(x); // where x is undefined
  • var y = null; Boolean(y); // y is null
  • var z = false; Boolean(z); // z is false
  • var a = (1/"hi"); Boolean(a); // a is NaN
